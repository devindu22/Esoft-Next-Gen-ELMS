# ESOFT Next-Gen ELMS Portal

## Overview
A modernized, student-centric E-Learning Management System (ELMS) interface designed to elevate the academic experience at ESOFT Metro Campus. This project replaces the legacy interface with a clean, high-engagement, light-themed UI utilizing glassmorphism design principles, fluid animations, and logical information architecture.

## Key Features

### 🔐 Authentication & Security
*   **Dual-Step Entry:** Seamless flow from a branded splash screen to portal credentials and Microsoft academic account verification.
*   **Secure Access:** Clean, professional login portals that prioritize student data security.

### 📊 Dashboard & Learning Hub
*   **Centralized Overview:** Real-time summary of student information, financial status, and notifications.
*   **Dynamic Learning Hub:** Easily filter between "All," "Ongoing," and "Completed" programs (e.g., Level 3 Diploma vs. HND in Computing).
*   **Progression Tracking:** Visual indicators for module completion and semester status.

### 📅 Smart Calendar
*   **Integrated Scheduling:** A fully interactive calendar grid featuring:
    *   Color-coded lecture schedules for multiple courses.
    *   Automated highlighting of Poya days, public holidays, and campus events.
    *   Presentation day tracking and assignment submission deadlines.
    *   Dynamic navigation to toggle between months.

### 📝 Submission Portal
*   **Integrated Workflow:** Dedicated assignment submission zones within course modules.
*   **Academic Honesty:** Built-in declaration checklist for BTEC/Academic integrity.
*   **File Handling:** Drag-and-drop support with real-time validation and submission receipt tracking.

### 💡 AI & User Experience
*   **AI Study Buddy:** An integrated virtual assistant for lecture summaries, navigation, and academic queries.
*   **UI/UX:** Soft, glassmorphic card design with a subtle ESOFT-branded linear gradient background.
*   **Responsive Footer:** Standardized footer containing copyright info and credit attribution.

## Tech Stack
*   **Frontend:** HTML5, CSS3 (Modern Flexbox & Grid layouts), JavaScript (ES6+).
*   **Design Assets:** ESOFT branding, custom CSS gradients, and modular components.
*   **Interactivity:** DOM manipulation for dynamic calendar rendering and filterable content.

## Getting Started
Since this project is a high-fidelity frontend prototype:
1. Clone the repository.
2. Open the main file (`index.html`) in any modern web browser (Google Chrome, Mozilla Firefox, or Microsoft Edge).
3. No backend server configuration is required to view the interactive UI/UX features.
